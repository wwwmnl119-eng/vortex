Telegram-style Android update starter.

Before build:
1. replace:
   https://YOUR_RENDER_URL.onrender.com
   with your real Render URL
2. replace:
   https://raw.githubusercontent.com/USERNAME/vortex/main/updates/version.json
   with your real GitHub raw URL

Build:
- open this folder in Android Studio
- wait for Gradle sync
- Build -> Build APK(s)
