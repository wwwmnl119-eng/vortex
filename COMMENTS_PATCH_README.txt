Добавлено:
- комментарии под постами канала
- кнопка "💬 Комментарии" под каждым постом
- отдельный экран комментариев
- отправка комментариев из интерфейса

Новые endpoints:
- GET /channel-comments/:postId
- POST /channel-comments/send

Как работает:
- в канале Vortex Offical каждый пост теперь открывает комментарии
- любой зарегистрированный пользователь может писать комментарии
- новые комментарии прилетают в реальном времени через socket.io
